## ✨ 核心功能

### 1️⃣ 群聊 CA 情报采集与展示
- 自动获取 **微信群 / Telegram 群聊** 中出现的合约地址（CA）
- 支持在以下终端实时查看：
  - 🌐 浏览器插件
  - 🌍 Web 网页
  - 📱 iOS 客户端
- 支持 **按群组查看** / **个人特别关注**
- 私域情报 & 高信噪比数据流，告别刷群焦虑
